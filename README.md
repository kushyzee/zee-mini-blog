# Zee Mini Blog

A simple and lightweight blog application built with React.

## Features

- Create, read, update and delete blog posts
- Responsive design
- Clean and modern UI
- Fast performance

## Installation

```bash
git clone https://github.com/kushyzee/zee-mini-blog.git
cd zee-mini-blog
npm install
npm start
```

## Technologies Used

- React.js
- TailwindCSS

## Project Structure

```txt
src/
  ├── components/
  ├── pages/
  ├── styles/
  └── utils/
```
